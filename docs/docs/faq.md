# FAQ

## 想用 TokenFlux 生图，该用哪个客户端？

推荐使用 [Cherry Studio](/docs/chatbot/cherry-studio)。

Cherry Studio 是一款支持多模型的桌面客户端，支持文生图模型，界面简洁，配置方便。接入 TokenFlux 后，在模型列表中选择支持生图的模型即可直接使用。

**快速开始：**

1. 按 [Cherry Studio 使用指南](/docs/chatbot/cherry-studio) 完成安装并接入 TokenFlux。
2. 在模型列表中选择支持生图的模型（如 `gpt-image-2` 等）。
3. 在对话界面发送提示词，即可生成图片。
