# 支持的国家和地区

生效日期：2025年10月30日

本文档说明 TokenFlux 当前支持商业 API 访问和相关服务的地区范围。

TokenFlux 仅向本页列明的国家、地区和领土提供服务。未列入本页的地区，默认不支持注册、购买、调用或继续使用相关服务；TokenFlux 也可能基于法律法规、上游政策、支付风控、制裁和出口管制要求调整可用范围。

如用户所在地、惯常居住地、注册地、实际控制人所在地或服务实际使用地不在支持范围内，用户不得通过 VPN、代理、代付、代注册、账号转让、接口转接或其他方式规避地区限制。
