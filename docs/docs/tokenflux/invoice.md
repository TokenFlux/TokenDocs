# 发票说明

TokenFlux 目前支持根据实际需求开具发票。不同类型的发票处理方式不同，请按需选择。

## 支持类型

- **Invoice**：请联系`daoge`开具。
- **大陆普通发票**：请联系`daoge`开具，并会收取 `6%` 服务费。

## 如何申请

请联系`daoge`，并提供以下信息：

1. 需要开具的发票类型：大陆普通发票或 invoice。
2. 需要开票的订单或充值记录。
3. 发票抬头、税号等必要信息。

确认信息后会协助处理。大陆普通发票因涉及人工处理和税务成本，处理时间可能会更长。
