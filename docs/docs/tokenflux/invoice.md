# 发票说明

TokenFlux 目前支持 `invoice`、大陆普通发票和大陆专用发票。不同类型的处理方式不同，请按需选择。

## 支持类型

- **Invoice**：使用 Stripe 支付会自动生成，无需联系管理员。
- **大陆普通发票**：请联系 `daoge` 人工处理，并会收取 `6%` 服务费。
- **大陆专用发票**：需要先签合同，且不能直接在网站在线支付，需要使用 `daoge` 提供的专门收款码付款。

## 开票主体

本站提供的发票抬头为：`深圳市萌凛科技有限责任公司`。

## 如何申请

如果需要大陆普通发票或大陆专用发票，请联系 `daoge`，并提供以下信息：

1. 需要开具的发票类型：大陆普通发票或大陆专用发票。
2. 需要开票的订单或充值记录。
3. 发票抬头、税号等必要信息。

确认信息后会协助处理。大陆专用发票需先完成合同和专门收款流程，请不要直接在网站在线支付。
