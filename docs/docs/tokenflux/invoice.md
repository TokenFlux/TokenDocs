# 发票说明

TokenFlux 目前支持 `invoice` 和大陆普通发票。不同类型的处理方式不同，请按需选择。

## 支持类型

- **Invoice**：使用 Stripe 支付会自动生成，无需联系管理员。
- **大陆普通发票**：请联系 `daoge` 人工处理，并会收取 `6%` 服务费。

## 开票主体

本站提供的发票抬头为：`深圳市萌凛科技有限责任公司`。

## 如何申请

如果需要大陆普通发票，请联系 `daoge`，并提供以下信息：

1. 需要开具的发票类型：大陆普通发票。
2. 需要开票的订单或充值记录。
3. 发票抬头、税号等必要信息。

确认信息后会协助处理。
