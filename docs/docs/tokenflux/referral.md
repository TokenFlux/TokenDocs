# 邀请返利

邀请好友使用 TokenFlux，每当被邀请的人首次消费后，邀请人会立即获得 `30🍥` 的返利。返利会直接发到邀请人的账户余额里。

## 如何找到邀请链接

1. 前往 [个人资料页面](https://tokenflux.dev/profile)。
2. 在页面底部找到邀请返利的相关链接和邀请码。
3. 复制邀请链接或邀请码分享给好友。

好友通过邀请链接注册并首次消费后，返利会自动发到邀请人的账户。

## 返利规则

- 被邀请的人首次消费后，邀请人会立即获得 `30🍥`。
- 返利直接发到账户余额，可用于后续消费。

## 相关入口

- [个人资料](https://tokenflux.dev/profile) — 查看和管理邀请链接
- [计费说明](/docs/tokenflux/billing) — 了解计费方式和购买选项
