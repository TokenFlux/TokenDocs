# 计费说明

本站采用 `🍥`（推理积分）作为计费单位。**模型广场显示的价格即为最终计费价格**，无需额外计算。

## 计费方式

本站支持两种购买方式：

- **订阅购买**：通常更便宜，有效期为 30 天。
- **按量购买**：按实际使用量计费，无时间限制。

前往 [充值/订阅页面](https://tokenflux.dev/purchase) 即可选择适合的购买方式。

## 价格如何计算

模型的计费包括三部分：输入、输出和缓存。每个部分都有对应的单价。

### 分组倍率

不同的 API Key 分组可能有不同的倍率。分组倍率是基于模型原价计算的，但模型广场显示的价格已经包含了倍率调整——那就是实际支付的价格。

**示例**

假设某模型的原价为：
- 输入：`2.5🍥`
- 输出：`15🍥`
- 缓存：`0.25🍥`

当分组倍率为 `1.3` 时，最终计费价格为：
- 输入：`3.25🍥`（2.5 × 1.3）
- 输出：`19.5🍥`（15 × 1.3）
- 缓存：`0.325🍥`（0.25 × 1.3）

这些调整后的价格就是模型广场显示的价格，也是最终支付的价格。具体的模型价格和分组倍率，请在 [模型广场](https://tokenflux.dev/models) 查看。

## 购买方式

### 订阅购买

订阅购买通常更便宜，但有效期为 30 天。

重复购买同一个订阅时：
- 不会并行新增一份同样的订阅
- 本质上是延长时间
- 后一次购买会在上一个订阅失效后生效

### 按量购买

按量购买无时间限制，按实际使用量计费。

前往 [充值/订阅页面](https://tokenflux.dev/purchase) 完成购买。

## 相关入口

- [邀请返利](/docs/tokenflux/referral) — 邀请好友获得返利
