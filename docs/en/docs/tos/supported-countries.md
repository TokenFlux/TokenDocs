# Supported Countries and Regions

Effective date: May 13, 2026

This document describes the regions where TokenFlux currently supports commercial API access and related services.

TokenFlux provides services only to the countries, regions, and territories listed on this page. Regions not listed here are unsupported by default for registration, purchase, calls, or continued service usage. TokenFlux may also adjust availability based on laws and regulations, upstream policies, payment risk controls, sanctions, and export-control requirements.

If a user's location, habitual residence, registration location, actual controller location, or actual service usage location is outside the supported scope, the user must not bypass regional restrictions through VPNs, proxies, proxy payments, proxy registration, account transfers, API relays, or other methods.
